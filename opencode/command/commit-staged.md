## Context

- Current git status: !`git status`
- Current git diff (staged changes): !`git diff --staged HEAD`
- Current branch: !`git branch --show-current`
- Recent commits: !`git log --oneline -10`

## Your task

Based on the above changes, create a single git commit.

Analyze the changes above and create a commit a comprehensive
but short commit message that focuses on the WHY NOT THE HOW of the changes.

If you are aware of a JIRA ticket ID that this work belongs to, prefix the
summary with the ticket id, i.e. "DE-123: My commit" instead of "My commit".
